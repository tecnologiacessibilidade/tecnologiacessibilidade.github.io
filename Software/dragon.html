Dragon Naturally Speaking

Dragon Speaking é um software, pago, de reconhecimento de fala que permite aos utilizadores falarem para um microfone enquanto que o software traduz as palavras para texto. Com este programa o utilizador não necessita de teclado. É um excelente software, mesmo para pessoas sem incapacidades.


Algumas funcionalidades:

  • Traduz fala para texto.
  • Pode ser usado em várias aplicações diferentes.

Existem vários produtos deste tipo, como por exemplo: SpeechMatics e Braina

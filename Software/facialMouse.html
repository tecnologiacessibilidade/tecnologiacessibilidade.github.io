EVA Facial Mouse

EVA Facial Mouse é um software, para dispositivos Android, que permite ao utilizador controlar o cursor através do movimento da cabeça. No entanto, não pode ser usado com outras aplicações que necessitem uso de câmara.


Algumas funcionalidades:

  • Mover o cursor com movimentos da cabeça.
  • A função clicar é ativada quando se deixa o cursor parado durantes um tempo.
  • Possui um menu que permite ao utilizador escolher uma ação.

Produto semelhante: Enable Viacam.

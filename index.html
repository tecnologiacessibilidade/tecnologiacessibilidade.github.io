Graças à petição pela Acessibilidade da Internet Portuguesa, iniciada em 3 de Dezembro de 1998 por Francisco Godinho e Ricardo Silva, foi possível criar regras para a Internet que garantissem acessibilidade independentemente da situação concreta em que se encontre cada utilizador.


Início

Sobre este Sítio Web

Este website, criado a pensar na acessibilidade dos seus utilizadores, tem como objetivo a divulgação de conselhos de Ergonomia, Produtos de Hardware e Produtos de Software que contribuem para melhorar a acessibilidade no computador de utilizadores com algum tipo de limitação.



Em Destaque: Noções de Acessibilidade

Acessibilidade Digital




Acessibilidade digital écaracterizada por um conjunto de recursos que possibilita a navegação, a compreensão e a interação de qualquer pessoa na internet (independentemente das suas dificuldades), sem a precisar de ter alguém a ajudar. Resumindo, uma internet acessível para todos!


Deve-se, portanto, promover o Design Universal: processo de criar produtos acessíveis para todos independemente se tem algum problema ou não. A aplicação do design universal diminui a necessidade de tecnologias de acesso ao computador, e torna o produto mais atrativo para todos, não só para pessoas com incapacidades.


Atualmente, estabele-se que:

  • Todos os sites Internet que disponibilizem exclusivamente informação estão obrigados a cumprir o nível 'A' das WCAG 2.0, a partir de 8 de fevereiro de 2013. O RNID recomenda mesmo para este tipo de sítios Web o nível 'AA'.
  • Todos os sites Internet que disponibilizem serviços online estão obrigados a cumprir o nível 'AA' das WCAG 2.0, a partir de 8 de fevereiro de 2013. O RNID recomenda mesmo o nível 'AAA' para este tipo de sítios Web.
